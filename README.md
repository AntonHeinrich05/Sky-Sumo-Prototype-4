# Sky Sumo – Prototype 4

This project is Prototype 4 from my Unity learning journey.
It is based on Unity’s Create with Code course.

## Features

- Player ball movement with forward/backward control
- Camera rotation around the center of the floating platform
- Movement always aligned with camera direction
- Multiple enemy types that try to push the player off the arena
- Wave system with increasing difficulty
- Boss enemy spawning after Wave 5, including minion spawns
- Knockback powerup for stronger enemy pushback
- Smash powerup that launches the player upward and slams down to blast nearby enemies
- Missile powerup that targets all active enemies
- Sound effects recorded with my own voice (pitched and processed)
- Background music from the Unity Asset Store (Free – Casual & Relaxing Game Music Pack)


## Purpose
This prototype focuses on:
- implementing simple gameplay mechanics
- enemy waves and increasing difficulty
- basic powerup systems
- camera‑aligned movement and knockback interactions
